Tutoriel et TP en HTML et CSS

Ceci est un site dans lequel nous montrons pas à pas comment utiliser des types de propriétés d'affichage Flex et grid. Pas de panique si vous ne connaissez rien à propos. Ca tombe bien! Apprenons ensemble.

Notre TP

Voici venu le moment tant attendu, celui de la pratique. Comme je le disais au début toute chose en html et CSS est une boîte, et nous avons appris à les ranger avec flex. Utilisons cet outils pour faire la page web de notre restaurant. Admirons ensemble le résultat.

Subdivisons le travail en trois parties

  • l'entête
  • le corps
  • le pieds de page

Pour commencer mettre tout le code dans une div#container, < div id="container" >.

A l'intérieur du body bien sûr. Puis ...

1. L'entête

Pour cette partie il vous faudra :

  1. HTML
    • Une balise < header > qui englobe toute l'entête
    • Une balise < nav > qui contiendra notre menu. Le menu se compose :
      • D'un titre, < h1 > ;
      • La liste de menu, Accueil, Plats du jour, Menus, Spécialités, A propos;
      • Un span qui contient deux boutons, < button >, l'un "commander" et l'autre "contacter;
        (< span > < button id="btn1" > Commander < /button > < button id="btn2" > < /button > < /span >)
    • Un conteneur, div.content, < div class="content" >. Ce cadre contiendra:
      • Une image (d'un plat) dans un div.photo. ( Pour insérer une image, utiliser < img src="assets/votre_imaage.jpeg" > en s'assurant que l'image est dans und dossier "assets" qui est dans votre projet de travil.) ;
      • Un div.content pour contenir le texte de l'entête, un titre et un paragraphe.
  2. CSS

      3. Utiliser convenablement les propriétés de flexbox.

    • Appliquer les proprietes flex sur chacun des container: header, nav, ul de liste de menu, content.
      • display: flex
      • flex-direction: convenable
      • justify-content: convenable
      • flex-basis pour la liste de menu
      • gap: valeurs convenable en rem, Comme 1.3rem ou 0.5rem
      • margin: auto
      Elles ne sont pas toutes obligatoires pour obtenir le résulat.
    • Définissez des dimensions fixes à votre image. Et changez la taille de police de vos titre et texte: font-size: 2.5rem;
    • Mettez des padding à vos conteneur de texte.
    • la couleur utilisée est: rgb(153, 12, 36)
