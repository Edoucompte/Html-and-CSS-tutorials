Tutoriel et TP en HTML et CSS

Ceci est un site dans lequel nous montrons pas à pas comment utiliser des types de propriétés d'affichage Flex et grid. Pas de panique si vous ne connaissez rien à propos. Ca tombe bien! Apprenons ensemble.

Les Prérequis de ce Tutoriel

Pour ce tutoriel vous n'aurez pas besoin de connaissances avancées, juste:

  1. Les bases de HTML (le code de base, les balises, leurs attributs id et class)
  2. Les bases de CSS (les selecteurs, les propriétés et quelques valeurs)
  3. Relier un ficher CSS à une page HTML
  4. Avoir un éditeur VsCode et savoir y utiliser le terminal en GitBash. Vous en faites pas, on va gérer.

C'est bon? On décolle !!!

Débutons au même niveau

Commençons par créer le projet dans lequel nous travaillerons dans vsCode.

  1. Ouvrons une nouvelle fenêtre vsCode
  2. Ouvrons un nouveau terminal
  3. Créons un dossier "Flex_Grid_Projet" et y ajoute nos codes
  4. Créons un fichier `index.html` et un ficher `style.css`
  5. Ecrivons le code de base HTML (avec ! sur vsCode) et relions le au style.css

Si les lignes de commande vous dérange, créez manuellement vos dossiers et fichiers.

Les types block et inline

Nous commencerons par remarquez deux types de balises en HTML :

Block

Ils occupent toute la ligne de la page. Ex: h1, p, div ..

Il n'y qu'une façon d'en être sûr. C'est codé!!!

  1. Ecrivez un titre dans votre code.
  2. Ajoutez lui une border et un background pour en être sûr

Inline

Ils occupent juste la plus necéssaire. Ex: a, span ..

Il n'y qu'une façon d'en être sûr. C'est codé!!!

  1. Ecrivez une liste avec ancres dans votre code.
  2. Ajoutez une border et un background à ses ancres li pour en être sûr

Et Voilà! C'est une très bonne remarque pour notre tuto.
C'est très moche pour un début, mais perseverons.
